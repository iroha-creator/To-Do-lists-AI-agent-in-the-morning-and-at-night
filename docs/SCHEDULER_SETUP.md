# Windowsタスクスケジューラー登録手順

常駐起動（スケジューラー）としてTodoAgentを運用するための手順です。

---

## 前提条件

- [セットアップ手順](./SETUP.md) が完了していること
- `.env` に必要な環境変数が設定されていること

---

## 手順

### 1. 起動スクリプトの作成

プロジェクトフォルダ内に `run.bat` を作成し、以下を貼り付けます。

```bat
@echo off
for /f "tokens=1,* delims==" %%A in (.env) do (
    if "%%A"=="AGENT_DIR" set AGENT_DIR=%%B
)
cd /d %AGENT_DIR%
call .venv\Scripts\activate.bat
python main.py
```

`.env` に `AGENT_DIR`（プロジェクトフォルダの絶対パス）を追記してください。

```env
AGENT_DIR=C:\Users\YourName\Documents\todo_agent
```

---

### 2. タスクスケジューラーを開く

`Win + R` → `taskschd.msc` → Enter

---

### 3. 新しいタスクを作成

右側ペインの「**タスクの作成**」をクリックします。

---

### 4. 各タブの設定

#### 「全般」タブ

| 項目 | 設定値 |
|---|---|
| 名前 | `TodoAgent` |
| セキュリティオプション | ユーザーがログオンしているときのみ実行する |

#### 「トリガー」タブ

「新規」をクリックし、以下を設定します。

| 項目 | 設定値 |
|---|---|
| タスクの開始 | ログオン時 |

#### 「操作」タブ

「新規」をクリックし、以下を設定します。

| 項目 | 設定値 |
|---|---|
| 操作 | プログラムの開始 |
| プログラム/スクリプト | `run.bat` のフルパス |
| 開始（オプション） | プロジェクトフォルダのフルパス |

#### 「条件」タブ

| 項目 | 設定値 |
|---|---|
| AC電源時のみ実行 | **チェックを外す**（ノートPC使用時に必須） |

---

### 5. 保存・動作確認

OKをクリックして保存後、タスク一覧に `TodoAgent` が表示されます。

右クリック →「**実行**」でターミナルが起動し、以下が表示されれば成功です。

```
📅 スケジューラー起動
   朝セッション: 08:00
   夜セッション: 21:00
   終了するには Ctrl+C を押してください
```

---

## 日常的な使い方

- PC起動・ログイン時にスケジューラーが自動で常駐起動します
- 設定した時刻になるとターミナルが前面に表示されます
- 会話終了後はターミナルの **✕ボタンで閉じる**だけでOKです
  - `Ctrl+C` はプロセスごと終了するため、次の時刻に起動しなくなります

---

## セッション時刻の変更

`.env` を編集するだけで反映されます。

```env
MORNING_HOUR=7
MORNING_MINUTE=0
EVENING_HOUR=22
EVENING_MINUTE=0
```
